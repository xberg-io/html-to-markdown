# html-to-markdown-rs - FFI (C/C++) Bindings

High-performance HTML to Markdown converter

## Installation

Link against `libhtml-to-markdown-rs_ffi` and include `html_to_markdown.h`.

See the build instructions in the main repository.

## Quick Start

```c
#include "html_to_markdown.h"

int main(void) {
    // See https://github.com/xberg-io/html-to-markdown for usage examples.
    return 0;
}
```
