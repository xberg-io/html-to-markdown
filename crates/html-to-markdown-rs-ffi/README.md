# html-to-markdown-rs - FFI (C/C++) Bindings

[![Bindings by alef](https://img.shields.io/badge/bindings%20by-alef%20%D7%90-007ec6)](https://github.com/kreuzberg-dev/alef)

High-performance HTML to Markdown converter

## Installation

Link against `libhtml-to-markdown-rs_ffi` and include `html_to_markdown.h`.

See the build instructions in the main repository.

## Quick Start

```c
#include "html_to_markdown.h"

int main(void) {
    // See https://github.com/kreuzberg-dev/html-to-markdown for usage examples.
    return 0;
}
```
