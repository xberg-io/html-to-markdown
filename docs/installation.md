# Installation

=== "Rust"
    ```toml
    [dependencies]
    html-to-markdown-rs = "3.0"```

=== "Python"
    ```bash
    pip install html-to-markdown```

=== "TypeScript"
    ```bash
    npm install @kreuzberg/html-to-markdown```

=== "Go"
    ```bash
    go get github.com/kreuzberg-dev/html-to-markdown/packages/go/v3```

=== "Ruby"
    ```bash
    gem install html_to_markdown```

=== "PHP"
    ```bash
    composer require kreuzberg/html-to-markdown```

=== "Java"
    ```xml
    <dependency>
        <groupId>dev.kreuzberg</groupId>
        <artifactId>html-to-markdown</artifactId>
        <version>3.0.0</version>
    </dependency>```

=== "C#"
    ```bash
    dotnet add package HtmlToMarkdown```

=== "Elixir"
    ```elixir
    {:html_to_markdown, "~> 3.0"}```

=== "R"
    ```r
    install.packages("htmltomarkdown")```

=== "C"
    Link against `libhtml_to_markdown`. See [releases](https://github.com/kreuzberg-dev/html-to-markdown/releases).

=== "WASM"
    ```bash
    npm install @kreuzberg/html-to-markdown-wasm```

## CLI

Install the command-line tool via Cargo:

```bash
cargo install html-to-markdown-cli
```

Or via Homebrew:

```bash
brew install kreuzberg-dev/tap/html-to-markdown
```
